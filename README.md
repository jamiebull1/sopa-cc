SOPA
====
Sopa es un "cookie-cutter" template para lanzar tu propio proyecto de Sopa 
en tu ciudad o barrio.

Todas las cosas son preparadas para empezar. Solo hay que rellenar los datos en 
el class `config.production.Config` y subirlo en tu alojamiento preferido.

#Heroku
El original sitio con este template es [Sopa Cáceres](http://www.sopacc.es),
alojado en el nuve de Heroku. Eso es una forma sencilla (y gratis) de lanzar tu
pagina. Necesita el variable de entorno `IS_PRODUCTION=True` y los detalles de
DNS y ya hasta!
